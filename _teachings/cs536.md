## CS 536: Computer Networks

**Role**: Teaching Assistant  
**Semester**: Fall 2023  
**Institution**: Purdue University

### Course Description

CS 536 is an undergraduate course covering fundamental concepts in computer networking, including:

- Network architecture and protocols
- TCP/IP stack
- Routing algorithms
- Network security
- Wireless and mobile networks

### Responsibilities

- Conducted office hours to assist students with course material and assignments
- Graded programming assignments and exams
- Assisted with lab sessions on network programming and protocol analysis
- Helped students debug socket programming assignments

### Skills Applied

- Network protocol analysis
- Socket programming (Python, C)
- Wireshark packet capture and analysis
- Teaching and mentoring
